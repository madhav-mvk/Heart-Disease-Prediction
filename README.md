# Heart-Disease-Prediction
The following objectives were defined for this project: 
• Train a supervised machine learning model on real patient data - not synthetic, not 
curated to be easy, but actual clinical records from the UCI Heart Disease Dataset. 
• Work with 1,025 patient records across 14 clinical features - and do the preprocessing 
properly, because a model is only as good as the data you feed it. 
• Three classification algorithms were selected - Logistic Regression, Random Forest, 
and SVM - and tested against the same data, because picking a winner without 
comparison is just guessing with extra steps. 
• Each model was put through a full evaluation - Accuracy, Precision, Recall, F1-Score, 
and Confusion Matrix - because a number that only tells you what went right isn't 
particularly useful in healthcare.
